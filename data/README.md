# Data

The BanglishRev dataset is not included in this repository due to size.

**Download from:**
[BanglishRev on arXiv](https://arxiv.org/abs/2412.13161)

**After downloading**, place the file at:
data/BanglishRev.json.gz

All notebooks read from this path by default via the DATA_PATH 
variable in the configuration cell.
